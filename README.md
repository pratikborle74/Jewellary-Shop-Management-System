# 💎 Jewellery Shop Management System  

A **Python-based Jewellery Shop Management System** designed to manage jewellery inventory, customer purchases, and sales analytics efficiently. This system includes features for **inventory management, sales tracking, automated billing, and data visualization** to streamline jewellery business operations.  

---

## 🚀 Features  

### 🏪 **Jewellery Inventory Management**  
✔️ Add, update, and delete jewellery items.  
✔️ Store and retrieve jewellery data using **CSV files**.  
✔️ Display a list of available jewellery products.  

### 💰 **Sales & Billing System**  
✔️ Process customer purchases with **automated invoice generation**.  
✔️ Calculate GST (18%) and additional taxes (5%) on purchases.  
✔️ Store and retrieve sales transactions from CSV.  

### 🧑‍🤝‍🧑 **Customer Management**  
✔️ Maintain customer records with **name, address, and phone details**.  
✔️ Save and load customer data for future transactions.  

### 🔐 **Admin Panel**  
✔️ **Secure authentication** for admin access.  
✔️ Manage jewellery inventory effectively.  
✔️ Visualize **most sold products** using a **bar graph**.  
✔️ Track sales trends with **sales-over-time graphs**.  

### 📊 **Data Visualization & Reports**  
✔️ Generate **bar graphs** for top-selling jewellery.  
✔️ Track **daily and historical sales trends**.  
✔️ Save detailed sales reports in CSV format.

## 🛠 Installation Guide
